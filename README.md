# Olanrewaju Akinola Portfolio

Personal portfolio website showcasing creative direction, technical projects, and AI governance research.

**Live site:** [elolamirewaju.github.io/Olanrewaju-Akinola-Portfolio](https://elolamirewaju.github.io/Olanrewaju-Akinola-Portfolio/)

## About

Creative Director and Technologist with 7+ years experience in broadcast media, brand storytelling, and emerging tech. Currently pursuing a PhD in AI Governance at the University of Manchester.

## Features

- Responsive design with light/dark theme toggle
- Interactive particle constellation background
- Glassmorphism UI elements
- Smooth scroll animations
- Video carousel for creative work
- Contact form with Formspree integration
- SEO optimized with Open Graph tags

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- CSS Custom Properties for theming
- Canvas API for particle effects
- Formspree for form handling

## Sections

- **Hero** - Introduction with key stats and CTAs
- **About** - Role switcher for different expertise areas
- **Skills** - Capabilities, tools, and key metrics
- **Experience** - Career timeline
- **Work** - Video showcase carousel
- **Technical Projects** - Web development portfolio
- **Research** - PhD and academic work
- **Testimonials** - Client and collaborator feedback
- **Contact** - Direct contact info and message form

## Local Development

Simply open `index.html` in a browser. No build step required.
